## ğŸš€ Quick Start

### Prerequisites
- Node.js v18+
- MongoDB
- Cloudinary Account
- Stripe Account

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/mern-multivendor-marketplace.git
cd mern-multivendor-marketplace
```

2. Install dependencies
```bash
# Backend
cd backend
npm install

# Frontend
cd ../frontend
npm install
```

3. Setup environment variables (see `.env.example` files)

4. Start the application
```bash
# Backend (Terminal 1)
cd backend
npm run dev

# Frontend (Terminal 2)
cd frontend
npm start
```
